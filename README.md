# Atendimento Pacientes CRUD

Projeto academico de um sistema web para atendimento de pacientes.

O objetivo final do projeto e apresentar um CRUD usando Maven, Spring Boot como apoio de configuracao e execucao, JSF na camada de visualizacao, JPA para persistencia, Servlet para uma funcionalidade auxiliar futura e arquitetura MVC.

## Tecnologias previstas

- Java 17
- Maven
- Spring Boot 3.x
- JoinFaces
- JSF
- JPA
- PostgreSQL
- Servlet
- Git

## Estrutura inicial

- `model`: classes de dominio e entidades JPA futuras.
- `repository`: interfaces de acesso a dados futuras.
- `service`: regras de negocio futuras.
- `controller`: controladores JSF futuros.
- `servlet`: servlet auxiliar futuro.

Nesta etapa o projeto contem apenas a estrutura inicial, sem entidade `Paciente`, sem telas de CRUD e sem API REST.
